# Reference — words, meanings & sources

Glosses for the spinner words, so the blog (and the curious) can see what each
one means and where it's verified. Scots words were checked against the
**Dictionaries of the Scots Language (DSL, dsl.ac.uk)** or **Wiktionary**; Fife
terms come from local dialect glossaries; football/culture references were
fact-checked against news and reference sources.

## fife — Fife & Dunfermline dialect and heritage

Sources: The Fife Post readers' glossary; Everything Edinburgh; verified history
for the heritage entries.

| Spinner word | Meaning / reference |
|---|---|
| Slaisterin' | working messily, making a slaister |
| Dichtin' | wiping (something) down |
| Loupin' | jumping; also "heaving / busy" |
| Skelpin' | smacking, hitting |
| Clipin' | telling tales, grassing someone up |
| Drookit | soaked through, drenched |
| Bealin' | raging, furious |
| Doutit | didn't understand / was unsure |
| Houkin' | the Fife "howk" — tattie (potato) picking; digging |
| Kennin' | knowing — "ken" is the Fife verbal tic |
| Pure Barry | (something) great, excellent |
| Neebsin' | neebs = mate/neighbour (mainly Fife) |
| Bunkerin' | the bunker = the kitchen worktop (distinctively Fife) |
| Doon the Lang Toun | Kirkcaldy, "the Long Town" |
| Eh ken | Fife phonetics — "eh" for I/my; "ken" = you know |
| Carnegie-ing | Andrew Carnegie, born Moodie St, Dunfermline 1835 — weaver's son, steel magnate, philanthropist |
| The Auld Grey Toun | Dunfermline's nickname |
| Glen-daunerin' | Pittencrieff Park ("The Glen"), gifted to the town by Carnegie |
| King-Robert-the-Brucing | Robert the Bruce, buried in Dunfermline Abbey (1329); "KING ROBERT THE BRUCE" carved round the tower |
| Up-the-Pars | Dunfermline Athletic FC, "The Pars" |
| Howkin' coal | West Fife coal-mining heritage (Valleyfield, Comrie, Culross) |
| Culrossing | historic royal burgh of Culross, near Valleyfield |

## scottish-football — legends, current squad & the patter

Player references fact-checked against ESPN/Sky 2026 squad coverage; legends are
well-documented Scottish footballing figures.

Legends: **McCoisting** (Ally McCoist) · **King-Kennying** (Kenny Dalglish) ·
**Denis-Lawing** (Denis Law, Ballon d'Or 1964) · **Baxtering** (Jim Baxter) ·
**Sounessing** (Graeme Souness) · **Bremnering** (Billy Bremner) ·
**Gemmilling** (Archie Gemmill, *that* 1978 World Cup goal).
Managers (all Scots): **Fergie-ing** (Sir Alex Ferguson) · **Steining** (Jock
Stein) · **Shankly-ing** (Bill Shankly) · **Busby-ing** (Sir Matt Busby).
Current: **McTominaying · McGinning · Robertson-overlapping** (captain) **·
Gilmouring · Tierney-ing · Clarke-ing** (Steve Clarke, manager). Plus the patter:
**Bicycle-kicking, Nutmegging, Stramashing, Worldie-hunting, Top-binsing,
Tartan-Army-ing, Hampden-Roaring, Lisbon-Lioning, Ya beauty**.

> Billy Gilmour is included as a Scotland regular even though he missed the 2026
> finals through injury (replaced by Tyler Fletcher) — a fact the build only
> caught by verifying the squad.

## scottish-culture — songs, films, places, food, sayings, landmarks

Songs: **Walkin'-500-Miles / Sunshine-on-Leithing** (The Proclaimers) ·
**Flower-of-Scotland · Caledonia-ing** (Dougie MacLean) · **Takin'-the-High-Road**
(Loch Lomond) · **Speed-Bonnie-Boating** (Skye Boat Song) · **Auld-Lang-Syning**
(Burns) · **Yes-Sir-I-Can-Boogie-ing** (Tartan Army anthem).
Films: **Freedom-shouting / Bravehearting** · **Choose-Life-ing** (Trainspotting).
Outdoors & places: **Munro-bagging · Nessie-hunting · Bonnie-Banking ·
Loch-Lomonding**.
Food & traditions: **Drammin' · Irn-Bruing · Haggis-chasing · Deep-frying ·
Ceilidhing · Skirling · Caber-tossing · Hogmanaying · Burns-Nighting**.
Sayings: **Keepin'-the-Heid · Haud-Yer-Wheesht · Yer-Bum's-Oot-the-Windae ·
Bilin'-Yer-Heid · Pure-Dead-Brilliant · Lang-May-Yer-Lum-Reek · Sair-Fechting ·
Haste-Ye-Back · No-Scotland-No-Partying · Wha's-Like-Us · Here-We-Go**.
Landmarks: **Painting-the-Forth-Bridge** (the idiom for a never-ending task —
fitting for a spinner) **· Forth-Crossing · Kelpie-watching · Falkirk-Wheeling ·
One-o'Clock-Gunning** (Edinburgh Castle) **· Castle-Rocking · Stirling-Castling ·
Wallace-Monumenting · St-Andrews-ing**.
